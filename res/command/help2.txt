Help message for AmpleProgrammingLanguage https://github.com/Kariaro/AmpleProgrammingLanguage

Usage: --compile <relative to paths> [options] -o <output>
       --run <absolute>

options:
    --help -h -?
                  display this help message

    --dir -d <absolute>
                  set the working directory

    --paths -p "<relative or absolute>;<2>;<3>; ... "
                  add source folder paths

    --output -o <relative or absolute>
                  the output file path

    --format -x <format>
                  the output format type
                  [spooky]
                  [x86]
                  [ir]
